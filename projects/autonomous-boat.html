Autonomous Boat Project

Freshman Year Introduction to Engineering Design Challenge

Arduino 3D Design Sensors Navigation Team Project

Project Media

Project Overview

As part of my Introduction to Engineering course at UCF, I collaborated with a team to design and build an autonomous boat capable of navigating a predefined course. This project marked my first hands-on experience with robotics and autonomous systems, laying the foundation for my interest in control systems and automation.

Learning Objectives

Engineering Design Process

Applied the engineering design process from concept to implementation, including iterative testing and refinement.

Technical Skills

Gained experience with Arduino programming, sensor integration, and basic control systems.

Team Collaboration

Worked effectively in a team environment, managing tasks and communicating technical ideas.

Technical Implementation

System Components

  • Arduino Uno microcontroller for system control
  • Ultrasonic sensors for obstacle detection
  • Servo motors for propulsion and steering
  • Custom 3D-printed hull design
  • Waterproof electronics housing

Design Challenges

  • Waterproofing electronic components
  • Balancing boat stability with maneuverability
  • Implementing reliable navigation algorithms
  • Power management and battery life optimization

Key Takeaways

  • Introduction to autonomous system design principles
  • Practical experience with sensor integration and control
  • Understanding of waterproof design considerations
  • Importance of iterative testing and refinement
  • Value of team collaboration in engineering projects
